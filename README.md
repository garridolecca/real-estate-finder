# Real Estate Finder

An ArcGIS Maps SDK for JavaScript application built with React for commercial real estate visualization and discovery.

**Live Demo:** [https://garridolecca.github.io/real-estate-finder/](https://garridolecca.github.io/real-estate-finder/)

## Tech Stack

- [ArcGIS Maps SDK for JavaScript](https://developers.arcgis.com/javascript/latest/) v5
- React 19 + TypeScript
- Vite 7
- Calcite Design System

## Packages

- [`@arcgis/core`](https://www.npmjs.com/package/@arcgis/core)
- [`@arcgis/map-components`](https://www.npmjs.com/package/@arcgis/map-components)
- [`@arcgis/charts-components`](https://www.npmjs.com/package/@arcgis/charts-components)
- [`@esri/calcite-components`](https://www.npmjs.com/package/@esri/calcite-components)

## Getting Started

```bash
npm install
npm run dev
```
